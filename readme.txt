Instructions for use:
-Run main.py to start program.

File menu
-Reset option restarts simulation. Altered setpoints will remain, allowing initial conditions to be set up.
-Exit option quits program. Program may freeze momentarily while shutting down.

Help menu
-About option displays credits

Left Panel
-Graph shows PID output from selected controller.

Right panel
-Top selection box allows set point of selected controller to be changed. Apply changes with "Set Setpoint" button.
-Plot Shown radio buttons switch the currently selected controller.
-Labels at bottom display current values of each controller.
